# Pack de Pós Instalação do Windows 10

Olá! Sabem, mesmo com toda a estabilidade do mundo, é porreiro ter uma forma rápida de colocar um sistema pronto, da forma mais automática possível. Isso no Linux, é feito com uns bash scripts e similares, o que torna qualquer instalação em frações de segundos. No Windows, dá para fazer algo igual, mas...limitações da minha parte, acredito. Mesmo assim, aqui apresento um montante de ficheiros que levo sempre depois de instalar o Windows 10. Vais poder encontrar coisas como alguns programas essenciais para o uso do computador, debloats do windows, o ativador do próprio e algumas mitadas que coloquei lá. Se seguires todos os itens que tenho, vais ter um sistema porreiro para tudo na tua frente. Como também quero experimentar fazer umas tabelas épicas, aqui vai uma lista do que tens lá dentro e os seus links atualizados, pois aviso épico em baixo.

Com o passar do tempo, alguns instaladores vão ficar datados, porque serão criadas novas versões. Portanto, recomendo que procures os novos .exe's de alguns dos programas.

# Limpar Computador Lento / Corrigir Erros do Computador / Eliminar Vírus do Computador

Hoje apresento uma ferramenta bastante poderosa para os casos extremos onde o computador está com performances horríveis ou similares, mas não temos a possibilidade de formatar o computador. Esta ferramenta, trata-se de um único script, que apresenta uma coletânea de scripts para a entrega do resultado desejado (Matrioska type shit). Ao certo, o que podemos esperar deste amiguinho? Senso sincero, não consigo acrescentar muito mais, já que nunca o utilizei o suficiente para descrever com exatidão, portanto, apenas baseio a minha descrição com o próprio conteúdo do site. Em suma, a função da ferramenta, é transformar um sistema Windows, muito mal tratado (com vírus, datado, muito lento) e o tratar para estar com saúde. Isto parece poderoso, mas não tomem isto como uma cura para todos os males. Basicamente, não levam isto como uma ferramenta para rodar todos os dias, ou algo parecido. Aliás, já que toquei nesse assunto, esta ferramenta consome muito tempo da vossa máquina. Podem esperar em média, umas boas 4-6 horas do vosso dia gastas com a ferramenta ligada. Bem, melhor do que eu, apenas a documentação. Antes de passar a ferramenta, recomendo a ler e tomar as dicas para o resultado desejado.

# Browsers

Eu já tentei ser equipa Firefox e eu compreendo os problemas que a base chromium trazem ao de cima, mas eu sou incapaz de não utilizar um. Por essa mesma razão, que eu uso browsers dessa mesma "arquitetura" como uma forma de consistência no meu trabalho. Por ser uma base bastante popular, existem muitas versões do Chrome por ai e olhem, eu já as usei quase todas. Comecei no clássico Chrome e num futuro, veremos no que estou. É bastante frustrante o mercado de browsers atuais, portanto, as minhas escolhas é mais por necessidade e "são os únicos" do que um gosto pessoal pelo produto.

  • Brave - De todos os daily drivers, o Brave é o meu escolhido. Ele é bastante simples e já bem configurado de fábrica com boas escolhas no quesito de privacidade e praticidade para qualquer usuário. Não gosto nada daquilo todo do "crypto" e como isso envolveu a empresa em escândalos relacionados ao descumprimento do que eles prometem fazer. Mesmo assim, continuo no uso. Desabilitei tudo o que consegui desse ramo, para ficar com a experiência mais simples possível dele. Recomendo apenas instalar algumas extensões, como qualquer browser e seguir jornada. Para este lado da segurança e privacidade online, recomendo o uBlock Origin, Noscript e Decenraleyes.
  • Ungoogle Chromium - Este é o browser que usaria 24/7, se não fosse pelo facto que ele é só focado na privacidade. Toda a questão da privacidade, é uma faca de dois gumes, já que em troca dela, perdemos muitas funcionalidades e similares. Para este browser, temos o fator mais importante, que é ao fechar o browser, perdemos qualquer tipo de logins que tínhamos feito e também a dificuldade de instalar extensões. Por esta mesma razão, que eu apenas uso este browser, quando quero fazer um trabalho que exige esse nível maior de segurança. Um aviso importante, este browser não tem qualquer tipo de servidor de atualização e por essa razão, ele não se atualiza sozinho. Portanto, eu mais que recomendo, tentarem o atualizar manualmente, quando acharem necessário. Isto é como forma de prevenir qualquer tipo de ataque numa falha de segurança, porque vocês estão a usar software desatualizado.
